Hexo的markdown添加图片

操作

  1. 设置根目录下_config.yml中的两个参数
    • post_asset_folder: true
    • permalink: :year/:month/:day/:title/
  2. 运行该命令npm install https://github.com/CodeFalling/hexo-asset-image --save
  3. 通过hexo new "Hexo的markdown添加图片"新建文章,此时同时生成一个与文章同名的目录,在此目录中放置图片,例如图片为tuzki.gif
  4. 在markdown中使用![兔斯基](tuzki.gif)显示图片

插件

  • hexo-asset-image
